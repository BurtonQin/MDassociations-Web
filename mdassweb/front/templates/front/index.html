MiRNA-Cancer Prediction

The web server mainly includes experimentally supported miRNA-Cancer associations, predicted miRNA-Cancer associations (RCNMN) and miRNAs took part in the cancer KEGG pathways.
The Browse records 45220 cancer-miRNA associations that have been verified. Users can query the known cancer-miRNA associations by miRNA entry or Cancer entry.

RCNMN and KEGG

The RCNMN is a Regression model based It on Cancer-cancer Network and MiRNA-miRNA functional similarity Network (RCNMN). It can be used to predict potential cancer-miRNA associations. The study analyzes the relationship between 2603 miRNAs and 134 cancers.
The KEGG include 15 cancer pathways. We integrate all miRNAs verified by experiment in to the above cancer KEGG pathways. User can take the enrichment analysis of miRNA in the cancer KEGG pathways.
